## Pasos para correr el proyecto
### con Docker
1. `docker build -t mis-turnos-medicos .`
2. `docker run -p 3000:3000 -d mis-turnos-medicos`

### sin Docker
1. Instalar Node v8.10.0 o superior.
2. `npm install`
2. `npm start`
